Fichiers Astro de remplacement pour les petites pages.

Ils utilisent les composants :
- WideImageBanner.astro
- PortraitCard.astro
- VerticalImageCard.astro
- SoftImageTriptych.astro
- CalendlyButton.astro
- ScrollReveal.astro

Principe :
- 1 à 2 images par page maximum, sauf triptyque léger.
- Animations : fade-in, slide-right, slide-left, zoom-in.
- Style cohérent avec soft-card, section-bg et les variables CSS du site.
